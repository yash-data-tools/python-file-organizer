# Python File Organizer

This project is a Python automation script that organizes files in a directory based on their file extensions.

## Features
- Automatically creates folders for specific file types
- Moves files into their respective folders
- Prevents unnecessary folder creation
- Uses built-in Python modules only (`os`)

## Technologies Used
- Python
- os module

## Use Case
This script is useful for cleaning up messy folders like Downloads, Desktop, or project directories.
