# Python File Organizer

This project is a Python automation script that organizes files in a directory based on their file extensions.

---

## Features
- Automatically creates folders for specific file types
- Moves files into their respective folders
- Prevents unnecessary folder creation
- Uses only built-in Python modules
- Interactive command-line input

---

## Technologies Used
- Python
- os module

---

## Use Cases
This script is useful for cleaning up messy folders such as:
- Downloads
- Desktop
- Project directories

---

## How to Run
1. Place the script (`main.py`) in the directory you want to organize  
2. Run the script using Python  
3. Enter the folder name and file extension when prompted  

---

## Example
Enter the folder name you want: text_files 
Enter the extension name you want: .txt 
Are you done sorting: y

---

## Notes
- The script will not create folders unless matching files exist
- The script ignores itself to prevent accidental movement
